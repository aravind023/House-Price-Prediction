# 🏠 House Price Prediction

## 📌 Overview
This project predicts house prices using Machine Learning algorithms.

## 📂 Dataset
The dataset contains house features such as area, bedrooms, bathrooms, and other attributes used to predict the selling price.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## 📈 Workflow
1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Model Training
6. Model Evaluation
7. House Price Prediction

## 📁 Files
- `HousePricePrediction.ipynb` – Complete notebook
- `data.csv` – Dataset
